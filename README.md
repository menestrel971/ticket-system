# 🤖 Automind HUB — Система обработки заявок (MVP)

Полный цикл обработки заявок:  
**Форма → FastAPI → Claude AI → SQLite → Telegram → Дашборд**

---

## 🚀 Быстрый старт (локально)

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Настроить переменные окружения
cp .env.example .env
# Открыть .env и заполнить ключи

# 3. (Опционально) Загрузить демо-данные
python seed.py

# 4. Запустить
uvicorn main:app --reload --port 8000
```

Открыть: http://localhost:8000

## 📦 Деплой на Railway

### Шаг 1 — Создать Telegram бота
1. Открыть Telegram → найти **@BotFather**
2. Написать `/newbot`
3. Дать имя боту (например: `AutomindHUB Bot`)
4. Скопировать **токен** (вида `7123456789:AAF...`)
5. Написать боту `/start` — он ответит
6. Открыть: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
7. Найти `"chat":{"id":XXXXXXXXX}` — это ваш **chat_id**

### Шаг 2 — Получить Anthropic API ключ
1. Зарегистрироваться на https://console.anthropic.com
2. Settings → API Keys → Create Key
3. Скопировать ключ (начинается с `sk-ant-...`)

### Шаг 3 — Деплой на Railway
1. Зарегистрироваться на https://railway.app (бесплатно, через GitHub)
2. New Project → Deploy from GitHub repo
3. Подключить этот репозиторий
4. В Railway → Variables добавить:

| Переменная | Значение |
|-----------|---------|
| `ANTHROPIC_API_KEY` | `sk-ant-...` |
| `TELEGRAM_TOKEN` | `7123456789:AAF...` |
| `TELEGRAM_CHAT_ID` | `123456789` |
| `PORT` | `8000` (Railway подставляет автоматически) |

5. Railway сам соберёт и запустит проект
6. Скопировать публичный URL (вида `https://xxx.railway.app`)

## 🤖 Как работает AI-классификация

При каждой новой заявке система отправляет содержание в **Claude API** с промптом, который просит определить:
- **Категорию**: техническая проблема / вопрос по оплате / консультация / жалоба / запрос функции / другое
- **Приоритет**: критический / высокий / средний / низкий
- **Обоснование**: почему именно эта категория и приоритет
- **Краткое резюме**: суть обращения в одном предложении
